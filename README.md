# 家庭防災設計 LP

## PDF保存手順

A4印刷用の設定は `index.html` に追加済みです。背景画像・背景色を残し、主要セクションやカードの途中で改ページされにくいようにしています。

1. `outputs/index.html` をブラウザで開く
2. ブラウザの印刷画面を開く
   - Windows: `Ctrl + P`
   - Mac: `Command + P`
3. 送信先で「PDFに保存」を選択
4. 用紙サイズを「A4」に設定
5. 余白は「既定」または「最小」を選択
6. 「背景のグラフィック」をオンにする
7. 必要に応じて倍率を `95%〜100%` の範囲で調整
8. PDFとして保存

## 印刷時の注意

ブラウザの仕様上、1ページに収まりきらない長いセクションは途中で改ページされる場合があります。カード、料金プラン、プロフィールなどの主要ブロックは途中で割れにくい設定にしています。
